# Simulation Results Images

This directory should contain the simulation result images organized as follows:

## File Naming Convention

For each simulation (numbered 01-20), you need three files:

1. **GIF file**: `sim_XX.gif` - The animation/GIF showing the simulation
   - Example: `sim_01.gif`, `sim_02.gif`, etc.

2. **Target Clarity Map**: `sim_XX_clarity.png` - The target clarity map image
   - Example: `sim_01_clarity.png`, `sim_02_clarity.png`, etc.

3. **Decay Field**: `sim_XX_decay.png` - The decay field image
   - Example: `sim_01_decay.png`, `sim_02_decay.png`, etc.

## Structure

- `sim_01.gif`, `sim_01_clarity.png`, `sim_01_decay.png`
- `sim_02.gif`, `sim_02_clarity.png`, `sim_02_decay.png`
- ... and so on up to `sim_20`

## Total Files Needed

- 20 GIF files (sim_01.gif through sim_20.gif)
- 20 Clarity map images (sim_01_clarity.png through sim_20_clarity.png)
- 20 Decay field images (sim_01_decay.png through sim_20_decay.png)

**Total: 60 image files**

## Layout on Website

Each simulation will be displayed in a card with:
- GIF animation on top
- Two images below (side by side): Target Clarity Map on the left, Decay Field on the right
- Two simulations per row
- Ten rows total (20 simulations)
